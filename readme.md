# Tensorflow Train Tools

### React + Flask + Tensorflow 

## How to Run
```
- Install dependency
----------------------
python <3.6
tensorflow > 2.0
pandas >1.0.3
flask > 1.1.2
flask-cors 3.0.8
----------------------
cd client
npm install
----------------------
pip install tensorflow pandas flask flask-cors
python server/app.py
```
